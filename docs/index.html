üîê Security & mTLS

Our security model relies on Mutual TLS (mTLS). Services must present a valid certificate signed by our internal CA to communicate.

GatewayController.java 
public Mono<ResponseEntity<String>> proxyRequest(Principal principal) {
    // 1. Validate Certificate Presence
    if (principal == null) {
        return Mono.just(ResponseEntity.status(401).body("mTLS required"));
    }

    // 2. Forward with Identity Headers
    return backendWebClient
            .method(method)
            .header("X-Client-CN", principal.getName())
            .retrieve()
            .toEntity(String.class);
}
